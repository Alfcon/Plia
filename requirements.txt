# =====================================================
# Plia (Pocket Local Intelligent Assistant) - Python Dependencies
# =====================================================
#
# Install all dependencies with:
#   pip install -r requirements.txt
#
# IMPORTANT: This project requires Python 3.11 or higher
#
# NOTE: Playwright is OPTIONAL. Plia's web search panel uses the
#   `ddgs` package (DuckDuckGo) and does not need Playwright.
#   Only install Playwright (uncomment in the Browser Automation
#   section below) if you plan to build custom agents that drive
#   a real browser. After installing the pip package, also run:
#       playwright install
#   to download the browser binaries.
#
# IMPORTANT: PyTorch is installed below with CUDA 12.4 support via
#   the --extra-index-url directive. If you are on a CPU-only machine,
#   remove the torch/torchvision/torchaudio lines and the index URL —
#   a CPU-only build will be pulled in automatically by transformers.
#
# IMPORTANT: The OpenAI package (see Web & API section below) is
#   installed by default, but Plia core (chat, voice, weather,
#   web search, briefing, planner, agents, reading files, etc.)
#   does NOT use OpenAI — everything runs locally on Ollama.
#
#   An OpenAI API key is only required for Agent Builder-generated
#   custom agents that opt-in to GPT-4o + Internet Search.
#   Get a key from platform.openai.com and set it in the Plia
#   Settings tab under "OpenAI API Key".
# =====================================================
